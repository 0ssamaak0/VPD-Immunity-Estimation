# VPD Immunity Estimation & Outbreak Estimation

## Project Pitch
Participants will work with an immunity estimation tool, **imuGAP**, and use the results to estimate outbreak distributions for vaccine-preventable diseases.

## Research Areas
- Data Science
- Estimation
- Measles
- Outbreaks
- Data Analysis

## Type of Data
- Local surveillance data (as available)
- Vaccination doses administered
- Coverage surveys, etc.

### Data Source
- Ideally your own data!
- Can also use U.S. measles data.

### Description and Status of Data
- Data quality may vary.
- U.S. data have been curated and cleaned.

## Software
- **imuGAP**

## Potential Research Questions
- How would changing vaccination inputs affect expected outbreak sizes or spatial distribution?
- What do we estimate missing coverage to be?
- Can we detect changes in vaccination patterns?
